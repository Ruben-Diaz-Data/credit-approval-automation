# Credit Approval Automation

Este proyecto simula un **sistema de aprobación de crédito** para una fintech de crédito al consumo.  
El objetivo es demostrar un flujo end-to-end de recepción de solicitudes, evaluación mediante reglas y automatización de decisiones.

---

## Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Crear un entorno virtual
3. Instalar las dependencias
4. Ejecutar la aplicación con:

```bash
uvicorn app.main:app --reload
